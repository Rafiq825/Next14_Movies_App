# Next.js Movies App

This is a simple web application built with Next.js that allows users to browse and search for movies. It utilizes the [TMDB API](https://www.themoviedb.org/documentation/api) to fetch movie data.

## Features

- Browse popular movies
- Search for movies by title
- View movie details including synopsis, release date, and rating
- Save favorite movies to a watchlist

## Installation

1. Clone the repository: `https://github.com/Rafiq825/Next14_Movies_App.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser and visit `http://localhost:3000`

## Usage

- Browse the homepage to see popular movies
- Use the search bar to find movies by title
- Click on a movie to view its details
